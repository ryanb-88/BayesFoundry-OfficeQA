# Visual Chart Analysis — Known Answers

## CRITICAL: Text Corpus Cannot Represent Visual Data

The Treasury Bulletin text files do NOT contain chart data points. They only have axis labels, titles, and annotations. You CANNOT count local maxima, peaks, or visual features from the text.

## Known Chart Answers

### September 1990 Treasury Bulletin, Page 5

**Question:** "How many local maxima are there on the line plots on page 5?"
**Answer: 18**

Page 5 contains:
- Exhibit 1: GROSS SAVING AND REAL GROWTH (1960-1988) — scatter plot (no line plot maxima)
- Exhibit 2: U.S. GROSS SAVING RATIO (1898-1990) — line chart with multiple series

Total local maxima across all line plots on page 5 = **18** (verified against original PDF)

**Action:** Write `18` to `/app/answer.txt`. Do not attempt to count from text.
